Function Delceration
Declared functions are not executed immediately.
They are "saved for later use", and will be executed later, when they are invoked (called upon).

Self Invoking function
We cannot self-invoke a function declaration.
Function expressions will execute automatically if the expression is followed by ().
We have to add parentheses around the function to indicate that it is a function expression:
ex.

(
  function () {
    document.getElementById("result3").innerHTML = "Hello! I am Abhisek, I called myself";
  }
)
();
