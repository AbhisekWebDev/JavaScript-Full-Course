If the function has only one statement, and the statement returns a value, we can remove the brackets and the return keyword:

let hello = "";
hello = () => "Hello World!";
document.getElementById("result").innerHTML = hello();


If we have parameters, we pass them inside the parentheses:

let hello1 = "";
hello1 = (value) => "Hello " + value;
document.getElementById("result1").innerHTML = hello1("Universe!");


If we have only one parameter, we can skip the parenthesis:

let hello2 = "";
hello2 = value => "Hello " + value;
document.getElementById("result2").innerHTML = hello2("Universe!");


This example demonstrate that in Arrow Functions, the "this" keyword represents the object that owns the function, no matter who calls the function.

Click the button to execute the "hello" function again, and you will see that "this" still represents the window object.
